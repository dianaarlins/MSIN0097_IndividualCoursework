# Predicting Late Deliveries in Brazilian E-commerce

## Overview
This project builds a machine learning model to predict whether an order will be delivered late using purchase-time information from the Brazilian Olist e-commerce dataset.

## Notebook
MSIN0097_CW.ipynb – main notebook containing data preparation, exploratory analysis, modelling, and evaluation.

## Files
- MSIN0097_CW.ipynb – main project notebook
- requirements.txt – Python libraries required to run the notebook

## Dataset
This project uses the Brazilian Olist e-commerce dataset.

Download it from:
https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce

Place the required dataset files in the same folder as the notebook before running it.

## How to run

1. Install the required libraries: pip install -r requirements.txt  
2. Open the notebook: jupyter notebook MSIN0097_CW.ipynb  
3. Run all cells to reproduce the analysis, figures, and results.
